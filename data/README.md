# Dataset Download Instructions

---

## Overview
The dataset used for this project is not stored in this repository due to size limitations and repository best practices.

---

## Dataset information
- **Title:** Comprehensive Diabetes Clinical Dataset (100k rows)
- **CSV title:** diabetes_dataset.csv
- **Size:** 6.14 MB
- **Link:** https://www.kaggle.com/datasets/priyamchoksi/100000-diabetes-clinical-dataset/data

---

## Download Instructions
1. Create, or log in to your Kaggle account.
2. Navigate to the dataset link above.
3. Download the `diabetes_dataset.csv` file
4. Place the CSV file in this `data/` folder inside your local project directory

---

## Important Notes
- Do not commit raw dataset files to GitHub.
- A Kaggle account is required to access and download the dataset.
